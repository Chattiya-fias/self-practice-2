- part 01 จะเป็นการสรุปและทบทวน traversing Nodes พร้อมทั้งแบ่งแยกการทำงานเพื่อความเข้าใจ รวมถึงลองเรียกใช้เพื่อหาจำนวนลูกด้วยวิธีที่แตกต่างกัน
- part 02 สรุปคำสั่งในการ select พร้อมทั้งแบ่งแยกรูปแบบ NodeList และ HTMLCollection รวมถึงลอง select แตกต่างกันดู และเนื่องจาก selectorAll 
มันจะได้ออกมาเป็น list จึงลอง query หาลูกทั้งใช้ method และใช้ index ตรงๆ
- part 03-05 question 3 ข้อที่อาจารย์ได้ให้โจทย์ในห้อง ซึ่งการแก้โจทย์ก็มีหลายวิธี จึงยกตัวอย่างเปรียบเทียบเพื่อศึกษาและทบทวนวิธีที่แตกต่างจากของตัวเอง
- part 06 เรียนรู้ถึงคุณสมบัติและลักษณะของ NodeList และ HTMLCollection ที่ถูกกล่าวถึงก่อนหน้านี้หลายครั้ง เพื่อความเข้าใจและการปรับใช้ในอนาคต
- part 07 ตอนเรียนในห้องตามไม่ค่อยทันเรื่อง Manipulating Nodes แต่พอมานั่งทบทวนเองก็รู้สึกว่าไม่ได้ยาก ทั้งไปเจอวิธี insertBefore ไปไว้หลังสุดซึ่งรู้สึกว่าเจ๋งดี
และทำให้ได้เรียนรู้หลากหลายรูปแบบ
- part 08 เป็นการทบทวนเนื้อหา Node Relationships ที่ได้เคยเรียนรู้ไปน่าจะ week ที่แล้ว เพื่อทบทวนผลลัพย์ที่ยังคงสับสน
- part 09-10 dataset and classList เป็นเรื่องที่เรียนไม่ทันเลยในห้อง เบื้องต้นในห้องก็ได้ศึกษาเองใหม่ไปพลางๆระหว่างทำ practice และศึกษาใหม่อีกครั้งใน self-practice
แต่ก็ยังคงติดเรื่อง event ที่ต้องศึกษาเพิ่มในเร็วๆนี้
