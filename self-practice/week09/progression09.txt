ไฟล์ self-practice09.js เป็นการทบทวนเนื้อหาบวกเรียนรู้ซ้ำ เรื่อง Event เนื่องจากมองว่า แม้การใช้ event จะง่าย แต่มีรายละเอียดยิบย่อยเยอะที่ทำให้งงหรือสับสนได้
เพราะเรียนในห้องก็เบลอๆ เอ๋อๆ ตามไม่ทันอยู่บ้าง โดยได้แบ่งออกเป็น 6 part ซึ่งแต่ละ part ก็จะมีคำอธิบายขณะ self-practice แนบไว้

- part 01 เรียนรู้ System Dialogs ซึ่งเป็นส่วนหนึ่งของการเขียนโปรแกรม ที่สามารถนำไปประยุกต์ใช้หลากหลาย รวมถึง Event
- part 02 recap concept ของ Event
- part 03 ทบทวนและ recap Event Propagation โดยในตอนแรกที่เรียนในห้อง ยังจำตัวเลขไม่ค่อยได้ เวลาอ่าน console ก็เลยงงๆ แต่พอได้ทบทวนก็ทำให้เข้าใจขึ้นมากๆ
- part 04 Event Handlers มองว่าไม่มีอะไรมาก เข้าใจง่าย คือมี add กับ remoce และหลักการเขียนตายตัว คือ ต้องมี three arguments
- part 05 ทำความเข้าใจ Event properties amd method ไปเบื้องต้นจากที่ได้เรียนกับอาจารย์
- part 06 Event types ซึ่งมีเยอะมากๆๆๆๆๆให้ได้เลือกใช้ ก็ลองใช้และดูการเปลี่ยนแปลงจากตัวอย่างโจทย์ของอาจารย์

และส่วนอื่นจะเป็นไฟล์ practice และ โจทย์ lesson3
- practice 01 ตอนทำในห้องทำไปได้แค่เช็คว่า input มีช่องไหนว่างไหม แต่ก็ไม่รู้ว่าถูกต้องหรือเปล่า จึงนำมาทำต่อใน self-practice จนเสร็จด้วยการงมกับตัวเองอยู่สักพัก
เพราะสับสนโจทย์ว่าต้องการให้ขึ้น missing ตอนกดปุ่มว่าไปเช็คว่ามี blur ไหม หรือให้ขึ้นเลยตอน blur เพราะแบบอย่างแรกก็พอทำได้ แต่อาจจะไม่ตรงโจทย์เท่าไหร่ เลยสรุปทำแบบที่สอง
- practice 02 โจทย์นี้ให้ลองใช้ Event types -> keyboard events เลยมองว่าง่ายกว่า practice 01 แล้วก็ศึกษาความแตกต่างของ keydown keypress keyup ไปในตัว
